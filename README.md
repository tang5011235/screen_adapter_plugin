# screen_adapter

屏幕适配插件

#
基于 [flutter_screenutil](https://github.com/OpenFlutter/flutter_screenutil)  开发的 仅用于android 和 ios 的屏幕适配 
用法于 [flutter_screenutil](https://github.com/OpenFlutter/flutter_screenutil) 基本一样。

# 使用
仅需一次初始化
```

Future<void> main() async {

ScreenUtil.init();

runApp(const MyApp());
}

```
